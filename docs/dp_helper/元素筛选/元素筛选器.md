



#  🔍️元素筛选器


## 🦆 用法  元素筛选和校验
-   在当前网页 按  F4   键，可以唤起 元素筛选器，默认抓取当前鼠标处元素的特征属性，  可以根据元素的属性  筛选出当前页面的元素，
-   再也不用在ide里调试 排查重复元素了
-   再次按  F4 键 可以关闭 元素筛选器
